# RE

规则:

^ 匹配字符串开头
$ 匹配字符串结尾
. 匹配任意字符串，除换行
[...] 里面的元素任意一个
[^...] 不在[]中的元素
re* 0个或多个
re+ 1个或多个
re? 0个或1个
re{n,} 匹配n个前面
re{n,m} 匹配n，m之间
a :单独匹配
(re) : 分组
\w ： 小写 匹配字母数组下划线 大写非
\s : 小写 匹配任意空白 等[\t\n\r\f]. 大写 非
\d ： 匹配然亦数组 等[0-9] 大写非数字
\n,\t,等，：匹配一个换行服，制表符，等
\1..\9：匹配第n个分组
[\u4e00-\u9fa5]：匹配中文

# 分类爬虫思想

  • url管理

    class URLManager(object):
	def __init__(self):
		self.new_url=[]
		self.old_url=[]
	
	def get_new_url(self):#获取一个url
		url=self.new_url.pop()
        self.old_url.append(url)
        return url
    
    def add_new_url(self,url):#添加一个url
    	if url not in self.new_url and url and url not in self.old_url:
    			self.new_url.append(url)

    def add_new_urls(self,urls):#添加多个url
    	for url in urls:
    			self.add_new_url(url)
           
    def has_new_url(self): #判断是否还有可以爬取的url
    	return self.get_new_url_size()>0?True:False
    	
    def get_new_url_size(self):#获取可以爬取的数量
    	return len(self.new_url)

	def get_old_url_size(self): #获取已经爬取的数量
		return len(self.old_url)

# scrapy⭕

是开源的网络爬虫框架，可导出json,csv,xml,支持xpath,css表达式选择器,正则来提取数据，不用导包了，跨平台，异步处理,可自动调节爬取速度。

spiders(解析)>scheduler(url管理)>downloader(下载器)>spiders(解析)>item pipeline(数据处理)

运行流程：

  • 引擎从调度器中取出url用于接下来的爬取
  • 引擎把url封装成一个请求Request传给下载器
  • 下载器下载资源，封装响应包Response
  • 爬虫解析Response
  • 解析成实体item，则交给实体管道进一步处理
  • 解析的url，则就交给调度器等待抓取。

包括的组件:

  • 引擎：scrapy
  • 调度器：scheduler
  • 下载器：dowwnloader
  • 解析：spiders ⭕
  • 管道管理：pipeline ⭕
  • 下载器中间件：Downloaderr Middlewares主要处理引擎和下载器之间的请求混合响应 ⭕
  • 爬虫中间件：spider Middlewares
  • 调度器中间件：scheduler Middlewares

from pymysql import connect
from scrapy.utils.project import get_project_settings
class MysqlPipeline(object):
    def open_spider(self, spider):
        # setting=get_project_settings()
        #连接mysql方法
        self.connectmysql()

    # 连接mysql方法
    def connectmysql(self):
        #编码写utf8
        self.conn=connect(user="root",passwd="hadoop",host="hadoop111",db="scrapy",charset="utf8",port=3306)
        self.cursor=self.conn.cursor()#连接对象

    def process_item(self, item, spider):  # 先建好表
        for index in range(15):
            sql = 'insert into movie(movie_name,movie_evaluate,movie_image_url,movie_type,movie_to_star,movie_time) values("{}","{}","{}","{}","{}","{}")'.format(item["movie_name"][index],item["movie_evaluate"][index], item["movie_image_url"][index], item["movie_type"][index],item["movie_to_star"][index], item["movie_time"][index])
            self.cursor.execute(sql)#执行
            self.conn.commit()#提交
        return item

    def close_spider(self, spider):
        self.cursor.close()
        self.conn.close()
        print("爬取成功")
