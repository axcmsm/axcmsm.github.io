# 搭建图床

图床就相当于一个图片服务器，用来存储图片的，然后需要用到图片注解贴上链接就可以。

选择图床的条件：

  • 可靠性
  • 安全性
  • 操作是否简单
  • 价格

重要的数据可考虑自建，或者考虑收费等情况

以下是收集到的免费图床：

七牛云 https://www.qiniu.com/ (opens new window) 快稳，需备案认证 免费10g

秋风图库 https://tu.slomoo.cn/ (opens new window)

麦克图床 https://macimg.com/ (opens new window)

路过图床 https://imgtu.com/ (opens new window) 推荐

公益图床 https://sbimg.cn/ (opens new window)

sm.ms https://sm.ms/ (opens new window) 推荐 5g

搭建图床的分类

  • 代码托管图床 : 码云、github等
  • 自建图床 :自己利用服务器搭建图床
  • COS :腾讯云 阿里云 等

# 我的选择：

# GitHub + jsDelivr + TinyPNG+ PicGo 打造稳定快速、高效免费图床

gitub的缺点就是国内访问速度慢,jsDelivr 可以弥补其缺点.

jsDelivr 是国外的一家优秀的公共 CDN 服务提供商，该平台是首个「打通中国大陆与海外的免费CDN服务」，无须担心中国防火墙问题而影响使用。官网：http://www.jsdelivr.com/ (opens new window)

搭建步骤:

  • 新建仓库

  • 生成token密钥

  • 下载picgo进行配置

  • 注意自定义域名:图片上传后，PicGo 会按照 自定义域名+上传的图片名 的方式生成访问链接，此处我们填写jsDelivr的CDN加速地址，格式：https://cdn.jsdelivr.net/gh/<用户名>/<仓库名>

  • 使用https://tinypng.cn/ (opens new window) (opens new window)压缩你要上传的图片（如图片已经很小或你有更好的压缩工具可省略这一步）

  • 在PigGo的上传区上传你的图片，到相册一键复制刚刚上传的图片URL即可
